# Barcode P2P Scanner (No Server)

Scan or type barcode on mobile → instantly send to laptop.

## 🚀 Features
- No backend required
- Works with barcode scanner devices
- Real-time communication (PeerJS)
- Clean UI

## 🧠 How to use

1. Open `laptop.html` on your laptop
2. Copy the generated ID
3. Open `mobile.html` on your phone
4. Paste ID and connect
5. Scan or type → auto send 🎉

## ⚠️ Notes
- Both devices need internet
- Works best on Chrome
